# Personal AI Synergy Assistant

A sophisticated personal task management system designed to optimize productivity through AI-driven correlation analysis and weighted priority scoring.

## Overview
This application is a personalized AI assistant that helps users manage daily activities and long-term goals. It leverages Large Language Models (Gemini API) to understand the relationships between different tasks and suggests the most efficient actions based on context, location, and success probability.

## Key Features
- **Long-term Goal Management:** Add and track major projects with success probability and correlation factors.
- **AI Correlation Engine:** Analyzes how different tasks (e.g., PhD studies vs. Thesis) support each other.
- **Dynamic Prioritization:** Calculates priority scores using weights for Urgency, Importance, and Synergy.
- **Privacy-First:** Your Gemini API key and data are stored locally in your browser's `localStorage`.
- **S25 Optimized:** Minimalist and responsive dark UI designed for high-end mobile experiences.

## How to Use
1. Clone this repository or host it via GitHub Pages.
2. Open the app and enter your Google Gemini API key.
3. Start by adding your long-term tasks in the "Long-term Goals" section.
4. Use the daily check-in to update the AI on your current location and activity.
5. Get real-time, intelligent suggestions on what to do next.

## Technology Stack
- HTML5 / CSS3 (Modern Flex/Grid)
- Vanilla JavaScript
